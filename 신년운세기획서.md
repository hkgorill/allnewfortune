# 📘 신년운세 서비스 기획서 (2025)

**서비스명(가칭):** 모두의 신년운세 (ALL NEW FORTUNE)
**수익모델:** 광고 기반 (카카오 AdFit · 네이티브 광고)
**디자인 방향:** 2025 감성적·미니멀 스타일 + 일러스트 중심 UI

---

## 1. 🔥 서비스 컨셉

* 2025년 트렌드인 **감성적·미니멀 디자인** 채택
* 입력 간소화: **생년월일만 필수**, 이름·성별·태어난 시간은 선택
* 4가지 운세 + 핵심 한마디 제공

  * 전체운
  * 금전운
  * 연애운
  * 건강운
  * 올해 꼭 명심해야 할 한 줄
* 독립된 서비스처럼 보이지만 **UX·디자인 시스템은 동일한 구조**
* 광고 수익 최적화 구조 포함

---

## 2. 🎯 서비스 목표

* 누구나 몇 초 만에 결과 확인
* 가볍고 직관적 UI → 이탈 최소화
* 연초/연말 트래픽 집중 활용
* 향후 MBTI/심리테스트 확장과 자연 연결
* 유튜브/Instagram 공유를 통한 유입 확장

---

## 3. 🧭 User Flow

### **1) 홈 화면**

* 감성적 일러스트 (달, 별, 새해, 소원 등)
* 주요 CTA: **신년운세 보기**
* 상단/하단에 테스트 메뉴 아이콘

  * 운세 / MBTI / 감정테스트 / 심리유형 등

---

### **2) 입력 화면**

* 라운드형 카드 UI
* 구성:

```
- 생년월일(필수) → Date Picker
- 이름(선택) → "입력 시 더 정밀한 결과 제공"
- 성별(선택) → Toggle(남/여/선택안함)
- 태어난 시간(선택) → Time Picker
```

* CTA: **운세 확인하기**

---

### **3) 로딩 화면**

* 감성적인 일러스트 애니메이션
* 문구 예:

  * “당신의 2025년을 분석하는 중…”
  * “새해 기운을 모으고 있어요 ✨”

---

### **4) 결과 화면**

상단: 사용자 정보 요약 (이름/생년월일/시간)
중앙 일러스트(캐릭터 또는 별자리 컨셉)

#### **결과 구성 (Markdown 기반)**

```
## 🌟 전체운
텍스트...

## 💰 금전운
텍스트...

## ❤️ 연애운
텍스트...

## 💪 건강운
텍스트...

---

### 📌 올해 꼭 명심해야 할 한마디
> "짧은 명언형 문구"
```

하단 기능:

* 공유하기 버튼
* 광고 배너
* 다른 테스트 추천 (MBTI, 감정유형 등 → 광고 영역 확장 가능)

---

## 4. 🎨 디자인 가이드라인

### **색상 (2025 미니멀 감성톤)**

```
Main: #FBD7D5 (파스텔 피치 핑크)
Secondary: #F7F3F4 (화이트 크림)
Highlight: #FFECD1 (소프트 오렌지)
Text Primary: #333333
Text Secondary: #777777
```

### **일러스트 스타일**

* 파스텔톤, 부드러운 그림
* 얇은 선, 라운드 형태
* Flat + 소프트 그라데이션

### **UI 컴포넌트**

* Card: border-radius 24px
* Shadow: 4~6px soft
* 네모형 버튼 → 라운드, 단순 아이콘 채용

## 5. ⚙ 기능 명세 (개발 용)

### **입력 파라미터**

```json
{
  "birthdate": "YYYY-MM-DD",
  "gender": "male | female | none",
  "name": "string | null",
  "birthtime": "HH:mm | null"
}
```

### **API 구조**

```
POST /api/fortune/year

{
  "birthdate": "...",
  "gender": "...",
  "name": "...",
  "birthtime": "..."
}
```

**Response:**

```json
{
  "overall": "string",
  "money": "string",
  "love": "string",
  "health": "string",
  "advice": "string",
  "illustration": "url"
}
```

## 6. 📱 화면 설계 (Wireframe 수준)

### **홈**

* 감성 일러스트
* [신년운세 보기]
* 관련 테스트 아이콘 목록

### **입력**

* 생년월일/이름/성별/시간
* CTA: 운세 확인하기

### **결과**

* 일러스트
* 전체운/금전운/연애운/건강운(Markdown)

## 7. 📡 광고 운영 구조

### 광고 위치

* 홈 하단 배너
* 결과 카드 사이 네이티브 광고
* 결과 끝단 배너
* 다른 테스트 추천 영역에 광고 삽입 가능

### 광고 형태

* AdFit 또는 Google AdMob Web
* 고정형, 스크롤형, 네이티브형 모두 호환
